# PHILOCHESS 0.1 code samples

This directory contains selected educational code samples derived from the
development of PHILOCHESS 0.1.

The samples illustrate specific technical milestones and have been adapted
for clarity and documentation purposes. They do not constitute the complete
PHILOCHESS source code and may differ from the current private implementation.

The published C++ samples are self-contained and can be compiled independently
with a C++17-compatible compiler.
